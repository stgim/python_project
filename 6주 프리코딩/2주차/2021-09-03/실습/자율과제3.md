### 교과목 학점 평가 시스템

<요구사항>

- 시험 점수(75%) + 과제 점수 (25%) = 총점으로 학점 계산(A, B, C, D)
- 시험 점수 입력 : 0 ~ 75점
- 과제 점수 입력 : 0 ~ 25점
- 정수만 입력 가능
- 범위를 벗어 나거나 잘못된 입력은 "경고 메세지" 출력

  | 총점         |  학점 |
  | :----------- | ----: |
  | 70 <= ~      | A학점 |
  | 50 <= ~ < 70 | B학점 |
  | 30 <= ~ < 50 | C학점 |
  | ~ < 30       | D학점 |

시험 조건
0 <= input <= 75
과제 조건
0 <= input <=25

`총합 25점 이하 일 경우 F학점`
